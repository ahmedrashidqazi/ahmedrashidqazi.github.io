Engineering

I've built a few production systems for clients — mostly AI-powered platforms that needed to actually work at scale. These projects involved full-stack development, microservices, and the usual messy realities of deploying ML models in production. Here's what I've shipped recently.

Fitness AI Coach
Built a SaaS platform for personalized fitness coaching (adults 40+). FastAPI backend with multi-provider LLM setup (Groq, OpenAI, Gemini) for workout generation, Next.js frontend, Stripe subscriptions, and proper observability with Langfuse and Sentry. Docker-deployed with PostgreSQL and Redis.

Stack: Python, TypeScript, FastAPI, Next.js, PostgreSQL, Docker, LangChain, Stripe
AI-Powered HR Platform
Architected an HR management platform for SMBs. Built a RAG-based chatbot using Qdrant vector DB for semantic search over documents, async document processing pipeline (PDF, DOCX, XLSX), AWS S3 storage, and integrated ticketing. FastAPI + Next.js, containerized with Docker Compose.

Stack: Python, TypeScript, FastAPI, Next.js, Qdrant, AWS S3, LangChain, Redis
